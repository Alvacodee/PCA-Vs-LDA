### Langkah-langkah

1.  **Clone Repository**

```bash
git clone https://github.com/Alvacodee/PCA-Vs-LDA.git
cd PCA-Vs-LDA
```
2. **UV sync**

Pastikan Python dan `uv` sudah terinstal.

```bash
# Install dependencies
uv sync
```

3. **RUN**

```bash
# Install dependencies
uv run main.py
```
